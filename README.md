# 👥 HR Analytics Dashboard

## 📌 Project Overview
The HR Analytics Dashboard is an interactive Power BI project developed to analyze employee attrition, workforce demographics, job satisfaction, and overall organizational performance. The dashboard provides valuable insights into employee trends and HR metrics, enabling data-driven workforce planning and decision-making.

---

## 🎯 Objectives
- Analyze employee attrition trends.
- Monitor key HR performance metrics.
- Identify workforce demographics and employee distribution.
- Generate actionable insights to support strategic HR decisions.

---

## 📂 Dataset
- **Dataset Format:** CSV
- **Domain:** Human Resources
- **Records:** Employee data containing demographic, job, salary, and attrition information.

---

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Power Query
- SQL
- Microsoft Excel

---

## 📈 Key KPIs
- Total Employees
- Attrition Rate
- Active Employees
- Average Age
- Average Monthly Income
- Job Satisfaction Score

---

## 📊 Dashboard Features
- Employee attrition analysis
- Department-wise workforce distribution
- Job role and education field analysis
- Salary and age group analysis
- Interactive filters and slicers for dynamic reporting

---

## 🔍 Key Insights
- Identified key factors contributing to employee attrition.
- Analyzed workforce distribution across departments and job roles.
- Evaluated employee demographics, salary trends, and job satisfaction.
- Provided insights to support strategic HR planning and employee retention.
